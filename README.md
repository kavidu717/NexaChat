# NexaChat - Real-time Secure Chat Application

NexaChat is a modern, real-time messaging application built with the MERN stack (MongoDB, Express, React, Node.js), featuring end-to-end encryption for messages, real-time communication via Socket.io, and a premium user interface using Tailwind CSS.

## 🚀 Key Features

* **Real-time Messaging:** Instant message delivery using Socket.io.
* **End-to-End Encryption:** Messages are encrypted using CryptoJS to ensure privacy.
* **Modern UI:** A sleek, mobile-responsive design with a dark theme and Tailwind CSS.
* **User Profiles:** View personal account details and manage settings.
* **Real-time Typing Indicators:** See when your friends are typing.
* **Secure Authentication:** JWT-based secure login and registration.


## 🛠 Tech Stack

* **Frontend:** React, Tailwind CSS, Vite, Socket.io-client.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB with Mongoose.
* **Real-time:** Socket.io.
* **Security:** CryptoJS (Encryption), JWT (Authentication).
